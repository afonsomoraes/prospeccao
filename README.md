# Guia de Organização - Prospecção Concept Marketing

Este diretório está organizado para que cada cliente prospectado tenha sua própria landing page acessível via subpasta.

## Estrutura Atual
- `index.html`: Redireciona para o site oficial da Concept Marketing.
- `_template/`: Pasta contendo os arquivos base para novos projetos.
- `biomed-podologia/`: Exemplo de projeto de cliente ativo.

## Como Criar uma Nova Página para Cliente

Para configurar um novo cliente (ex: `cliente-exemplo`):

1. **Duplique** a pasta `_template`.
2. **Renomeie** a cópia para o nome do cliente (use letras minúsculas e hífens, ex: `cliente-exemplo`).
3. **Edite os arquivos** dentro da nova pasta:
   - `index.html`: Altere textos, cores (Tailwind classes) e logo.
   - `assets/`: Adicione fotos e ícones do cliente.
4. **Envie para o servidor/GitHub**: Assim que a pasta estiver no repositório, ela ficará disponível em:
   `lp.conceptmarketingdigital.com.br/cliente-exemplo`

## Dicas de Prospecção
- Use o nome do cliente na URL para gerar autoridade imediata.
- Personalize a seção "Hero" com a dor específica que você identificou no negócio dele.
- Mantenha o CTA (WhatsApp) sempre visível.
