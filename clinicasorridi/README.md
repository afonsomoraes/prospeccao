# Clínica Sorridi Indaiatuba - Website

Website institucional de alta conversão desenvolvido para a Clínica Sorridi, unidade Indaiatuba. Focado em SEO local, performance mobile e conversão via WhatsApp.

## 🚀 Estrutura do Projeto

- `/`: Home Page (`index.html`)
- `/servicos/`: Listagem de especialidades e páginas detalhadas
  - `aparelho-ortodontico/`: Foco principal do projeto
  - `clareamento-dental/`
  - `implante-dentario/`
  - `facetas-e-lentes/`
  - `estetica-facial/`
- `/sobre/`: História e equipe da clínica
- `/contato/`: Informações de localização, telefones e formulário
- `/politica-de-privacidade/`: Conformidade LGPD
- `/assets/`: CSS customizado, JS para cookies e imagens (WebP/AVIF)

## 🛠 Tecnologias Utilizadas

- **HTML5 Semântico**
- **Tailwind CSS (via CDN)** para estilização ágil
- **Vanilla JavaScript** para gestão de cookies e LGPD
- **JSON-LD (Schema.org)** para SEO de LocalBusiness e MedicalProcedure

## 📈 SEO e Conversão

- Metatags Open Graph configuradas para compartilhamento em redes sociais.
- Sitemap e Robots.txt gerados.
- CTA flutuante de WhatsApp presente em todas as páginas.
- Design Mobile-First.

## ⚖️ Conformidade LGPD

O site inclui um banner de consentimento de cookies persistente via `localStorage`, garantindo que o usuário seja informado sobre a coleta de dados básicos de navegação.
